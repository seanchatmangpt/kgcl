## What You'll Learn

By completing these tutorials, you will:

1. **Install and configure** KGCL in your development environment
2. **Understand the basics** of knowledge graphs and RDF
3. **Create workflows** using YAWL Workflow Control Patterns
4. **Build hooks** that automate knowledge graph operations
5. **Run and debug** workflows using the CLI
